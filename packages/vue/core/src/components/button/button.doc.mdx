# Button

Button 按钮组件，用于触发操作或提交表单。支持多种变体（默认、普通、描边、文本、图标）、尺寸、加载状态、水波纹效果，基于 Ark UI 的 `ark.button` 原语实现。

## ARK UI Link

Button 基于 [Ark UI](https://ark-ui.com) 的 `ark.button` 原语实现，根组件的 props 会通过 `useForwardExpose` 透传到底层元素。

- **相关文档**: [Ark UI Vue Factory](https://ark-ui.com/vue/factory)
- **说明**: 根组件完全支持 Ark UI 对应原语（`button`）的 HTML 属性，可传递任意原生 HTML 属性（如 `data-*`、`aria-*` 等）。
