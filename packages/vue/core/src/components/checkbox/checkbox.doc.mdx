# Checkbox

Checkbox 复选框组件，用于单项勾选、混合态（`indeterminate`）展示与多选组场景。基于 Ark UI Checkbox 构建，支持主题尺寸配置与分槽位样式定制。

## ARK UI Link

Checkbox 对应 Ark UI 的 Checkbox 组件，根组件 props 会通过 `useForwardProps` 透传到 Ark UI Root。

- **相关文档**: [Ark UI Checkbox (Vue)](https://ark-ui.com/vue/docs/components/checkbox)
- **说明**: 根组件完整兼容 Ark UI Checkbox Root 的核心能力（如 `checked`、`defaultChecked`、`name`、`required`、`invalid` 等），并额外提供 `label`、`theme`、`ui` 便捷配置。
