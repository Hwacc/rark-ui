# Collapsible

Collapsible 折叠面板组件，用于在有限空间中展开/收起内容。该模块提供 `Collapsible`、`CollapsibleTrigger`、`CollapsibleContent` 以及便捷封装 `ReadMore`，支持主题尺寸配置与触发器/UI 样式扩展。

## ARK UI Link

`Collapsible`、`CollapsibleTrigger`、`CollapsibleContent` 基于 Ark UI Collapsible 组件实现，根组件 props 会通过 `useForwardProps` 透传到底层 Ark UI 组件。

- **相关文档**: [Ark UI Collapsible (Vue)](https://ark-ui.com/vue/docs/components/collapsible)
- **说明**: `Collapsible` 根组件完整支持 Ark UI Collapsible Root 的 props（如 `defaultOpen`、`disabled`、`collapsedHeight` 等），并额外支持 `theme/class` 扩展。
