# DatePicker

DatePicker 日期选择器，用于单日期或日期范围选择。组件支持日/月/年视图切换、禁用日期规则、事件监听与主题尺寸配置，适用于表单录入和筛选场景。

## ARK UI Link

DatePicker 基于 [Ark UI](https://ark-ui.com) 的 `useDatePicker` 和 `DatePicker.RootProvider` 实现，根组件 props 透传 Ark UI 的 `DatePickerRootBaseProps`，并在外层集成 Rark UI 主题能力。

- **相关文档**: [Ark UI Vue Date Picker](https://ark-ui.com/vue/docs/components/date-picker)
- **说明**: 根组件支持 Ark UI DatePicker 的核心能力（如单选/范围、视图控制、日期不可用规则、受控状态等），可直接复用 Ark UI 的参数约定。

常用事件（emits）：`focusChange`、`openChange`、`valueChange`、`viewChange`、`update:modelValue`、`update:open`、`update:view`、`update:focusedValue`。

复合子组件补充：

- `DatePickerControl`、`DatePickerContent` 不提供 `ui` crafts，支持基础结构与 `class`。
- `DatePickerDayView` 支持 `weekDayType`：`'short' | 'long' | 'narrow'`。
- `DatePickerMonthView` 支持 `monthType`：`'short' | 'long'`。

## UI Configuration

本组件不提供 `ui` props。DatePicker 的样式建议通过 `theme`（含 `crafts`）以及各子组件的 `class` 进行定制。

组件主要 props：`modelValue`、`selectionMode`、`isDateUnavailable`、`numOfMonths`、`open`、`view`、`focusedValue`。
