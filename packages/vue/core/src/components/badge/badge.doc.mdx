# Badge

Badge 徽章组件，用于展示状态、计数或提示信息。支持多种变体（默认、次要、圆点）和尺寸，可渲染为 `div` 或 `sup` 元素。

## ARK UI Link

Badge 基于 [Ark UI](https://ark-ui.com) 的 `ark.div` 和 `ark.sup` 原语实现，根组件的 props 会通过 `useForwardProps` 透传到底层元素。

- **相关文档**: [Ark UI Vue Factory](https://ark-ui.com/vue/factory)
- **说明**: 根组件完全支持 Ark UI 对应原语（`div` / `sup`）的 HTML 属性，可传递任意原生 HTML 属性（如 `data-*`、`aria-*` 等）。

## UI Configuration

本组件不提供 `ui` props。Badge 为单一根元素组件，无需通过 `ui` 配置多 slot 的样式；样式定制请使用 `theme` 和 `class`。

组件主要 props：`variant`（变体）、`as`（渲染元素）、`asChild`（复合组件）、`class`（自定义样式）。
