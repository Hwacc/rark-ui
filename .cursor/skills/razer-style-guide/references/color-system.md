# Razer 颜色系统

## 主色调

- **组件主色**：`#44d62c` (Razer 绿色)
- **组件高亮色**：`#73e161` (浅 Razer 绿色)
- **背景主色**：`#111111` (深色背景)
- **背景二级色**：`#222222` (较浅的深色背景)

## 文字颜色

- **文字主色**：`#000000` (黑色 - 用于浅色背景)
- **背景上的文字主色**：`#ffffff` (白色 - 用于深色背景)
- **背景上的文字二级色**：`#909090` (灰色 - 用于深色背景上的次要文字)

## 使用指南
- 组件主色 (`#44d62c`) 用于：
  - 激活状态
  - 选中状态
  - 悬停边框
  - 焦点指示器
  - 主要操作
- 组件高亮色 (`#73e161`) 用于：
  - 主要元素的悬停状态
  - 次要高亮
- 背景主色 (`#111111`) 用于：
  - 主容器背景
  - 卡片背景
- 背景二级色 (`#222222`) 用于：
  - 嵌套容器
  - 提升的表面
- 根据背景适当使用文字颜色：
  - 白色 (`#ffffff`) 用于深色背景上的主要文字
  - 灰色 (`#909090`) 用于深色背景上的次要文字、标签、描述
