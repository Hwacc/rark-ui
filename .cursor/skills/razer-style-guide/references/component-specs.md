# Razer 组件规范

## 组件状态

### 默认状态
- 背景：透明或 `#111111` (背景主色)
- 边框：`#909090` (文字二级色)
- 文字：`#ffffff` (深色背景上的文字主色)

### 悬停状态
- 边框：`#44d62c` (组件主色)
- 背景：透明（未选中时）或 `#73e161` (选中时)
- 文字：`#ffffff` (文字主色)

### 激活/选中状态
- 背景：`#44d62c` (组件主色)
- 边框：`#44d62c` (组件主色)
- 文字：`#ffffff` (文字主色) 或 `#000000` (浅色背景上的文字主色)

### 焦点状态
- 边框：`#44d62c` (组件主色)
- 轮廓：2px 实线 `#44d62c`

### 禁用状态
- 透明度：50% (0.5)
- 边框：`#909090` (文字二级色)
- 背景：透明
- 文字：`#909090` (文字二级色)
- 无悬停效果

## 组件尺寸

### 超小 (xs)
- 字体大小：12px
- 内边距：4px 8px
- 高度：24px
- 边框圆角：2px

### 小 (sm)
- 字体大小：14px
- 内边距：6px 12px
- 高度：32px
- 边框圆角：3px

### 基础 (base)
- 字体大小：16px
- 内边距：8px 16px
- 高度：40px
- 边框圆角：4px

### 大 (lg)
- 字体大小：18px
- 内边距：10px 20px
- 高度：48px
- 边框圆角：6px

## 间距系统

### 间距大小
- xs: 4px
- sm: 8px
- base: 16px
- lg: 24px

### 内边距大小
- xs: 4px
- sm: 8px
- base: 16px
- lg: 24px

### 外边距大小
- xs: 4px
- sm: 8px
- base: 16px
- lg: 24px

## 边框圆角

- xs: 2px
- sm: 3px
- base: 4px
- lg: 6px

## 阴影

- 默认：`0 2px 4px rgba(0, 0, 0, 0.3)`
- 悬停：`0 4px 8px rgba(0, 0, 0, 0.4)`
- 焦点：`0 0 0 2px rgba(68, 214, 44, 0.3)`

## 字体

### 字体系列
- 主要：Roboto（或系统默认无衬线字体）
- 次要：系统默认

### 字体粗细
- 正常：400
- 中等：500
- 半粗：600
- 粗体：700

### 行高
- xs: 1.2
- sm: 1.4
- base: 1.5
- lg: 1.6
