# Razer 尺寸指南

## 尺寸档位
- 分别设有四个尺寸档位: `xs`,`sm`,`base`,`lg`
- 根据不同的尺寸档位, 都有一套**预设尺寸**作为指南, 用于最基本参考

## 预设尺寸

### xs
- **font size**: 12px
- **line height**(最小): 1.25
- **border size**: 1px
- **border radius**: 2px
- **padding x**(最小): 6px
- **padding y**(最小): 4px

### sm
- **font size**: 14px
- **line height**(最小): 1.25
- **border size**: 2px
- **border radius**: 2px
- **padding x**(最小): 8px
- **padding y**(最小): 6px

### base
- **font size**: 16px
- **line height**(最小): 1.25
- **border size**: 2px
- **border radius**: 2px
- **padding x**(最小): 10px
- **padding y**(最小): 8px

### lg
- **font size**: 18px
- **line height**(最小): 1.25
- **border size**: 2px
- **border radius**: 4px
- **padding x**(最小): 12px
- **padding y**(最小): 8px

## 其他尺寸
- **shadow**: 15px
- **shadow md**: 10px
- **shadow sm**: 5px
- **outline**: 2px

## 注意
- **预设尺寸**仅作为基本量的参考, 具体的尺寸需求将以以下需求为准:
  1. `references/details`细粒度指南中的需求
  2. .pen文件中对应的Context中的需求
  3. 用户输入的prompt中的具体需求
- 在设计多档位尺寸时, 必须保持尺寸档位之间的比例关系,一致性,连续性
